# 《视传杀》在线发布说明

这个版本是纯网页游戏，可以直接发布成在线链接。当前形态是“单机牌局 + AI 对手”，玩家打开链接即可游玩；如果要做成多人在线房间对战，需要下一阶段增加服务器、房间、实时同步和断线重连。

## 最快发布方式

1. 使用 `outputs/shichuansha-online.zip`。
2. 上传到支持静态网站的平台。
3. 不需要构建命令，不需要安装依赖，入口文件就是 `index.html`。

## 推荐路线

- Netlify Drop：把压缩包或解压后的文件夹拖进去，生成一个在线链接。
- Cloudflare Pages：创建 Pages 项目，上传静态文件，构建命令留空。
- GitHub Pages：新建仓库，上传本目录全部文件，在 Pages 设置里选择 `main` 分支和根目录。

## 发给别人玩

发布成功后，把平台生成的网址发给别人即可。对方只需要浏览器，不需要安装软件。

## 后续多人在线升级

多人在线版建议增加：

- 创建房间、加入房间、房间码。
- 玩家座位、身份分配、准备状态。
- 出牌、选目标、结算的实时同步。
- 掉线重连和观战。
- 服务端校验，防止玩家改前端作弊。
